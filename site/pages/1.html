What do all these files mean?

Explaining the Cynthia file tree.  

Cynthia file tree



  • cynthia folder
    • main.js Cynthia itself
    • clean-cyn.tar.gz Stores a clean CynthiaConfig
    • package.json Tells NPM how to start and install Cynthia
    • package-lock.json Tells NPM what it has downloaded
    • README.md The Cynthia Readme file
    • node_modules created by NPM, stores dependencies of Cynthia
    • .env
    • cynthia_config
      • modes
        • default.jsonc
          • Sets the default environment and theme on root page of fresh Cynthia setupd
        • strawmelonjuice-cynthia.jsonc
      • styles
        • cynthia.css
        • default.css
      • templates
        • default.handlebars
        • cynthia-docs.handlebars
    • site
      • assets
        • svg
          • home.svg
          • cynthia.svg
      • pages
        • index.md
      • published.jsonc
